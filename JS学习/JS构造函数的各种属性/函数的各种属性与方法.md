1. **关于函数的各种属性 与 方法**

   1. ```js
      function User(name) {
      	var name = name; //私有属性
      	this.name = name; //公有属性
      	function getName() { //私有方法
      		return name;
      	}
      }
      User.prototype.getName = function() { //公有方法
      	return this.name;
      }
      User.name = 'Wscats'; //静态属性
      User.getName = function() { //静态方法
      	return this.name;
      }
      var Wscat = new User('Wscats'); //实例化
      ```

   2. **私有属性、私有方法**

      - 只能在函数内部作用域中使用，即  <font color=blue> 私有属性只有 私有方法才能使用</font>

   3. **公有方法、公有属性**

      - 只有  实例化对象（即 new 出的实例） 才能使用，<font color=red> 公有方法不能调用私有属性 与方法</font>
      - 公有方法 可以使用     公有属性

   4. **静态属性   与   静态方法**

      -   这些 怎么形成  就  怎么使用，静态方法 不能调用其他数据（除了 传参进来的数据）

2. **对于  构造函数 有无返回值 **

   1. **一般构造函数不应该有返回值，若有返回值，事实上   返回的就是  构造函数实例出来的实例**
      1. **没有返回值   **                                         
         -   var dd =  new  Demo() ;   dd  就是  Demo  的实例
   2. **构造函数有 返回值** 
      1. 返回值 为基本类型                                      
         - **与   没有返回值一样  **                                                               
      2. 返回值为  引用类型 ；当使用 new 创建实例的时候 ，实际是得到这个  引用类型
      3. **特殊情况：  返回  this ， 就是返回   构造函数的  当前实例化对象（就是构造函数的实例）**  

3. **<font color=red>注意！！！ </font>**

   - **因为  构造函数  就是  普通函数。所以  ` new Demo()`  和  `  Demo()`  的区别，必须要清楚,特别是 有返回值的时候  **
     - **new Demo( ) :  创建一个实例，有返回值   就根据上述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            的  分析**
     - **Demo( ) ：  执行函数，return 什么就得到什么；return  this ，即返回 执行该函数的所在环境作用域**
